## ğŸ“– Overview
The **Career Mentor App** is an interactive Streamlit application designed to guide students and early-career professionals in building actionable career roadmaps. It combines profile inputs, skill-based recommendations, roadmap generation, GitHub/LinkedIn feedback, and job matching into a single platform.

---

## âœ¨ Features
- **Profile Builder**: Capture education, skills, interests, and weekly availability.  
- **Recommendation Engine**: Suggests projects and learning paths based on skills and career interests.  
- **Roadmap Generator**: Converts recommendations into 2-week sprints with estimated hours, resources, and validation criteria.  
- **Profile Feedback**: Provides actionable GitHub and LinkedIn improvement tips.  
- **Job Matching**: Matches student skills with job postings and highlights missing skills.
